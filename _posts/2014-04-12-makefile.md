这是最基本的makefile命令，target会自动查询其依赖文件(depA，depB)    
寻找依赖文件的显式规则，尝试其符合的隐式规则，执行相应的命令进行编译

{% highlight make %}
target: depA depB
    command
    command
{% endhighlight %}

多个target会进行分解，下面两个是等价的

{% highlight make %}
targetA targetB: depA depB
    command
    command
{% endhighlight %}

{% highlight make %}
targetA: depA depB
    command
    command
targetB: depA depB
    command
    command
{% endhighlight %}
